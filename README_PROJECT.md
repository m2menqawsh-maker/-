# Co'Ma - نظام المحاسبة الجماعية

## 🎯 نظرة عامة
نظام محاسبة متكامل لإدارة مساحات العمل المشتركة (Coworking Spaces) مع دعم كامل لتخزين البيانات في Excel.

- ✅ إدارة المخزون والمشتريات
- ✅ نظام القروض والديون
- ✅ توزيع الأرباح على الشركاء

### 📊 التخزين

- ✅ سهولة الوصول والتعديل
- ✅ ترحيل من JSON القديم
- ✅ نسخ احتياطي تلقائي

### 👥 إدارة المستخدمين

## 📦 نظام Excel الجديد

### المزايا
1. **سهولة الوصول**: افتح البيانات في Excel أو Google Sheets مباشرة
2. **التنظيم**: كل جدول في ورقة منفصلة واضحة
3. **المشاركة**: سهولة إرسال الملف للمحاسب أو الشريك
4. **التعديل**: إمكانية التعديل اليدوي على البيانات
5. **الاحترافية**: مظهر أكثر احترافية

### كيفية الاستخدام
1. شغّل التطبيق
2. اختر "Excel Storage"
3. اختر ملف Excel موجود أو أنشئ جديد
4. ابدأ العمل! ✨

### ترحيل من JSON
إذا كنت تستخدم النظام القديم:
1. اختر "ترحيل من JSON إلى Excel"
2. حدد ملف JSON القديم
3. اختر موقع ملف Excel الجديد
4. سيتم الترحيل تلقائياً!

## 🔄 الترقية من JSON إلى Excel

### خطوات سريعة:
1. افتح `index.tsx`
2. استبدل:
   ```typescript
   import { initFileStorage } from './fileStorage';
   ```
   بـ:
   ```typescript
   import { initExcelStorage } from './excelStorage';
   ```

3. استبدل في كل الملفات:
   - `fileGet` → `excelGet`
   - `fileSet` → `excelSet`
   - `fileRemove` → `excelRemove`

4. شاهد `docs/QUICK_START.md` للتفاصيل الكاملة

## 🐛 حل المشاكل

### المشكلة: البيانات لا تُحفظ
**الحل**: 
- تأكد من عدم فتح ملف Excel في برنامج آخر
- تحقق من صلاحيات الكتابة على الملف

### المشكلة: "File System Access API not supported"
**الحل**:
- استخدم Chrome أو Edge
- سيعمل تلقائياً مع localStorage كبديل

### المزيد من الحلول:
راجع `docs/FAQ.md`
